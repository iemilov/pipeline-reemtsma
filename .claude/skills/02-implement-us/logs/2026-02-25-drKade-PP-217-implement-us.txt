## Implementation

### Root Cause
pTA_TrainingsCarousel.js extracted PTA_Order__c into a `rec.order` property but never called `.sort()` on the mapped array. The overview page (pTA_TrainingsList) already sorted correctly.

### Fix Applied
File: force-app/main/default/lwc/pTA_TrainingsCarousel/pTA_TrainingsCarousel.js

1. Added `.sort(this.compare)` to the end of the `.map()` chain in the `trainingList` wire handler
2. Added `compare(a, b)` method: `return a.order - b.order;`

Pattern follows pTA_BrandsCarousel (which already sorts correctly).
