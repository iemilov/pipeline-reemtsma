## Expense Categories
- travel (Reisekosten) — 19% VAT
- hotel (Übernachtung) — 7% VAT
- meals (Verpflegung) — 19% VAT
- transport (Fahrtkosten) — 19% VAT
- flatrate (Pauschale) — 19% VAT, unit: DAY
- other (Sonstige Auslagen) — 19% VAT

## Manual Steps Required
1. Create R2 bucket: wrangler r2 bucket create crm-attachments
2. Run migrations: npm run migrate:local && npm run migrate:remote
3. Configure expense limits in Settings page (optional)
